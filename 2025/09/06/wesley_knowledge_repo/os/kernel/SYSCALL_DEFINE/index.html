__前缀: 在 linux 内核源码中, 或者说在 C 语言的习惯中, 如果变量有前缀 __ 往往指代底层功能函数, 即少封装, 少检查, 底层, 内部用的含义

宏的使用中,

  • ## 是字符串连接符,
  • __VA_ARGS__ 用于在 可变参数宏(variadic macro)中表示"传入的全部可变参数"
  • ... 表示宏函数可以接受任意数量的 arg，由 __VA_ARGS__ 进行捕获
  • __SC_DECL 用来把一组“类型 + 形参名”的成对参数，拼成函数形参列表
