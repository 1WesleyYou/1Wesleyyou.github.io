  • 矢量场线

    • 基本形式

        1. h(x,y)y′+g(x,y)=0h(x,y)y' + g(x,y) = 0h(x,y)y′+g(x,y)=0
        1. h(x,y)dy+g(x,y)dx=0h(x,y)dy + g(x,y)dx = 0h(x,y)dy+g(x,y)dx=0

    • 思路

      • 首先变为方向场 y′=(1−gh)y' = \begin{pmatrix}1\\\frac{-g}{h}\end{pmatrix}y′=(1h−g​​)

      • 伸长为 $G =\begin{pmatrix}-h\g\end{pmatrix} $

      • 变为法向矢量场 G⊥=(gh)G^\perp = \begin{pmatrix}g\\h\end{pmatrix}G⊥=(gh​)

        • 我们知道对势能方程的求导得到的是梯度向量，是垂直于等势线，由于这里我们已经求了垂直矢量场，所以这个积分得到的势能方程就是原矢量场的积分曲线

      • 这样我们积分得到 U=∫G⊥U = \int G^\perpU=∫G⊥ ，或言之我们构造势能方程，分别关于 x,yx,yx,y 求偏导得到 g,hg,hg,h 就好了

    • 积分参数 integrating factor

      • 构造函数 MMM 只是关于 xxx 或 yyy 的函数

      • 对原解 (gh)\begin{pmatrix}g\\h\end{pmatrix}(gh​) 乘以参数得到 通解 (MgMh)\begin{pmatrix}Mg\\Mh\end{pmatrix}(MgMh​)

        • 本质上就是reduction of reduction

      • 由于垂直矢量场存在势能方程，因此场旋度为 0 ⇒\Rightarrow⇒ 交叉求导相等

        • ∂y(Mg)=∂x(Mh)\partial_y(Mg) = \partial_x(Mh)∂y​(Mg)=∂x​(Mh)
        • Myg+Mgy=Mxh+MhxM_y g + Mg_y = M_xh + Mh_xMy​g+Mgy​=Mx​h+Mhx​, 若 MMM 是只关于 xxx 的函数，则My=0M_y = 0My​=0
        • Mxh=M(gy−hx)M_x h = M(g_y - h_x)Mx​h=M(gy​−hx​)
        • 积分得到等式 ln⁡M=∫gy−hxh\ln M = \int \frac{g_y - h_x}{h}lnM=∫hgy​−hx​​ （不定积分）
        • 因此我们有积分解 M=e∫gy−hxhM = e^{\int \frac{g_y - h_x}{h}}M=e∫hgy​−hx​​
        • 通解势能方程 ∫x(Mg+c(y))=∫y(Mh+c(x))\int_x (Mg+c(y)) = \int_y (Mh+c(x))∫x​(Mg+c(y))=∫y​(Mh+c(x))

    • 与隐函数定理的结合

      • 二阶隐函数定理结论: 对于隐函数 fff 有构造显函数 y=g(x)y = g(x)y=g(x), f(x,y)=f(x,g(x))f(x,y) = f(x,g(x))f(x,y)=f(x,g(x)), g′=−(Dyf)−1⋅Dxfg' = -(D_yf)^{-1}\cdot D_xfg′=−(Dy​f)−1⋅Dx​f

      • 我们将定义 p:=Dxf,q:=Dyfp: = D_xf, q:=D_yfp:=Dx​f,q:=Dy​f

      • 故我们有 y′=g′(x)=−pqy' = g'(x) = -\frac{p}{q}y′=g′(x)=−qp​ , 这个显函数就可以有积分方程（势能方程） Φ(x,y)=∫x(Mp+c(y))=∫y(Mq+c(x))\Phi(x,y) = \int_x (Mp+c(y)) = \int_y (Mq+c(x))Φ(x,y)=∫x​(Mp+c(y))=∫y​(Mq+c(x))

        • 首先这个积分方程就是一个势能方程，也是一个implicit function

        • 通过给定初始值我们有等式 $\Phi (x_0,y_0) =c $

          • 其实不给定初值问题我们得到的从 yyy 角度是一组平行曲线
          • 在 implicit function 中我们通过 拉格朗日乘子问题 知道这也是一组平行的曲线

        • 很多时候我们不需要找到积分因子，只需要求一个基本解就完了

          • 这个基本解可以通过分解得到
          • pdx+qdy=0pdx+qdy = 0pdx+qdy=0 直接积分得到 Φ(x,y)=∫pdx+∫qdy\Phi(x,y) = \int pdx + \int qdyΦ(x,y)=∫pdx+∫qdy

      • 所以我们的微分方程的解可以通过转变为积分势能函数代入初值解决隐函数限制最终得到一个可分解的结果

        • 为什么要这样做？

          • 微分方程的解是具有严格限制的，尤其是对已经给定的 y′=−ghy' = -\frac g hy′=−hg​ 使用积分曲线会更加直接方便
