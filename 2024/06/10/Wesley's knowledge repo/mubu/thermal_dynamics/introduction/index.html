  • 热力系统 system:

    • 系统 system：被选为研究对象的一定量的物质或者区域

    • 外界 surrounding：系统外的物质或者区域，能和系统发生质能交换

    • 边界 boundary：系统和外界之间的分界面

      • 特点：可以固定不动，也可以变形或者位移；可以实际存在，也可以假想

    • 系统的分类：

      • a. 只有能量交换没有物质交换：闭口系统 closed system; 兼有物质能量交换 open system

        • 注：区分关键在于是否有物质穿过了边界，而不是质量守恒（min=moutm_{in}=m_{out}min​=mout​ 不属于closed）

        • open 例子：compressor压缩机；turbine涡轮；nozzle管口

          • 涡轮

            • 作用：压缩空气使得氧气浓度提升，是内燃机燃烧效率更高，用于汽车和飞机发动机内部
            • 原理：和压气机对称使用，涡轮通过类似蜗牛外壳的装置将管道横截面积逐渐变小，根据伯努利连续性原理，汽车排出的废气会在这个管道中不断加速，在最中心位置经过一个风扇转变为轴的转动动能，而轴的另一端与压气机相连，压气机工作原理与涡轮完全相反，管道横截面积逐步增大，且转动的风扇通过离心力将空气甩到压缩机的内壁上形成高压（也叫离心泵），最终通向内燃机

        • open也是有边界的：例如nozzle管口，其两端虽然是开的但是也可以视作是两个边界

      • b. 无热量交换 绝热系统 isothermo

        • 注：区分关键是有无热量穿过边界 open 系统也可以是绝热系统，例如保温性能良好的管子

      • c. 无能量物质交换 孤立系统 isolate system

        • 注：一切相互作用给都发生在系统内部
        • 注： 孤立系统一定是绝热，绝热不一定孤立

      • if 能量 && 物质 : （true表示有交换）

        • open sys (常称为控制体积系统)

        • elif 能量 && not 物质 :

          • closed (CM)

        • elif 物质 && not 热量 :

          • 绝热系统

        • elif not 物质 && not 能量 :

          • 孤立系统

    • 系统的划分：不同部分归为一个系统时的分类不同

  • 比体积 v=Vm\mathcal{v} = \frac{V}{m}v=mV​ 单位 m3kg\frac{m^3}{kg}kgm3​ 是密度的倒数

  • CM: control mass即系统的质量不变，不等于closed system； CV 即控制容积，体积可变（如开口系统）

  • 性质 property

    • 强度性质与广度性质 intensive and extensive

      • intensive property: 数值上与体系中物质的量无关的性质，即不具有“部分加和性质”

        • 例：温度，密度，温度

      • extensive property: 数值上与体系中物质的量相关的性质

        • 例：质量，体积

      • 区分方式：将一个系统分为两个部分，不变的是int，变化的是ext

      • specific：对extensive性质求单位质量量

        • 例：specific volume v=Vmv = \frac{V}{m}v=mV​
        • 例：specific total energy e=Eme = \frac{E}{m}e=mE​

  • 平衡 equilibrium

    • 热平衡 thermal equilibrium: 一个系统内各部分温度相等 （强调空间）

    • 机械平衡 mechanical equilibrium: 一个系统内部各部分压强关于时间不发生变化 （时间）

    • 相平衡 phase equilibrium: 各相质量不随时间变化

    • 化学平衡 chemical equilibrium: 化学成分随时间不发生变化

    • 状态原则 state postulate: 一个可压缩系统的状态至多需要两个独立intensive变量描述

      • 可压缩系统 simple compressible system：没有电磁力、引力、动作、表面张力作用的系统
      • 独立 independent：当一个变量发生变化时另一个变量可以保持不变

    • 恒流 steady flow：在一个系统内某一点的性质不随时间发生变化

      • 宏观表现：整个系统的质量、体积、能量保持不变
      • 应用：涡轮、泵、冷凝器

  • 过程和循环 path and cycles

    • 过程 process：系统从一个平衡到另一种平衡称为 process

    • 路径 path：process改变的方式称为路径

    • 准静态过程 quasi-static：系统在变化过程中无限接近于一个平衡状态，可以视为恒平衡形式

      • 工程上考虑准静态过程的原因：准静态过程的能量利用率最高且易于分析

    • 学术词：

      • isothermal 恒热
      • isobaric 恒压
      • isochoric/ isometric 恒定体积
