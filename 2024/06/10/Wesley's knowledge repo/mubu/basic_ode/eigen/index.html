4. 特征值与特征方程

  • 特征值 eigenvalue 与特征向量 eigenvector

    • Lv=λvLv = \lambda vLv=λv

    • 本征空间: Vλ={v∈V:Lv=λv}V_\lambda =\{v\in V : Lv = \lambda v\}Vλ​={v∈V:Lv=λv}

      • 几何重度 dim⁡Vλ\dim V_\lambdadimVλ​
      • 本征空间的另一种表述法 ker⁡(L−λ∘id⁡)\ker (L - \lambda \circ \operatorname{id})ker(L−λ∘id)
      • 每个本征空间只对应一个 λ\lambdaλ 值

    • 引理: 如果存在特征值集 $\lambda_1 \cdots \lambda_n \in \mathbb{F} $ 互不相等, 则 {v1,⋯ ,vn}\{v_1, \cdots, v_n\}{v1​,⋯,vn​} 是一组独立集合

      • 人话：特征值不同 ⇒\Rightarrow⇒ 本征空间互相线性无关

    • 推论：

      • (iii) 一个 Fn→Fn\mathbb{F}^n\to \mathbb{F}^nFn→Fn 的 endomorphism 至多有 nnn 个不同的特征值
      • (iiiiii) 一个 nnn 维度空间若有 nnn 个不同的特征值， 则其有 n 个线性无关的特征向量 $v_1, \cdots,v_n $ 就是空间的一组基

  • 特征方程characteristic polynomial

    • (A−λ∘id⁡)x=0(A - \lambda \circ \operatorname{id}) x = 0(A−λ∘id)x=0 有解 $\Rightarrow $ det⁡(A−λ∘id⁡)=0\det(A-\lambda \circ \operatorname{id}) = 0det(A−λ∘id)=0

    • 特征方程 p(λ)=det⁡(A−λ∘id⁡)=0p(\lambda) = \det(A - \lambda \circ \operatorname{id}) = 0p(λ)=det(A−λ∘id)=0 本质上是一个关于 $\lambda $ 的 nnn 次多项式

      • 特征方程是针对每一个 λ\lambdaλ 均不相同的
      • 代数重度 p(λ)=0p(\lambda) = 0p(λ)=0 的解的个数

    • 代数基本定理：在 C\mathbb{C}C 数域下的多项式 p(x)=Σi=0naixi=0p(x) = \Sigma_{i=0}^n a_ix^i = 0p(x)=Σi=0n​ai​xi=0 一定会有 nnn 个解

      • 推论: 在 C\mathbb{C}C 上的特征方程的代数重读之和为 nnn
      • 推论: 对任意矩阵 A∈Mat(n×n;C)A\in Mat(n\times n;\mathbb{C})A∈Mat(n×n;C) 至少有一个复数特征值（最不理想就是 唯一特征值）

  • 几何重度和代数重度的关系

    • 对于某个特定的特征值λ\lambdaλ, 几何重度小于等于代数重度

      • 等于: 对任意特征值均成立，则矩阵的行列式不为 0，矩阵可以对角化

        • 表述上是 几何重度的和为 nnn
        • 也就是 V=VλiV = V_{\lambda_i}V=Vλi​​ 的直和

      • 小于: 不能对角化

        • 但是可以上三角化

  • 自伴算子 self-adjoint: A∗=AA^* = AA∗=A

    • adjoint A∗=AˉTA^* = \bar{A}^TA∗=AˉT

    • 引理: LLL 是一个线性映射的自伴算子 ⇒\Rightarrow⇒ LLL 的所有特征值是实数

      • 推论: 自伴算子 LLL 至少有一个实数特征值（代数基本定理 + 自伴算子的性质）

    • 映射的不变域 invariant

      • 对某个线性映射 L∈L(V,V)L\in \mathcal{L}(V,V)L∈L(V,V), 我们有子空间 U⊂VU\subset VU⊂V, u∈U⇒Lu∈Uu\in U \Rightarrow Lu\in Uu∈U⇒Lu∈U

        • 人话：映射前后均在子空间里面 UUU

      • 引理: 对于不变域 UUU, 若 LLL 是自伴算子，则在全空间 VVV 下的 U⊥U^\perpU⊥ 也是一个LLL 下的不变域

        • 一般直接说的话是 L∗L^*L∗ 的不变域

  • 映射扩展和映射压缩

    • 映射压缩 restriction

      • L∣U:U→VL|_U: U\to VL∣U​:U→V, L∣Uu=LuL|_Uu = LuL∣U​u=Lu

        • 人话：本来是全空间映射 V→VV\to VV→V， 现在将自变量约束在 UUU 下，则像空间也只在 UUU 下

          • 大endo下存在小endo

    • 映射扩展 extension

      • L~:V→V\tilde{L}: V\to VL~:V→V， L~v=Lu\tilde{L}v = LuL~v=Lu

        • 人话：对于自变量定义子空间 UUU, 扩展映射满足在 UUU 上不变， 在 V \ UV\ \backslash\ UV \ U 空间下 L~v=0\tilde{L}v = 0L~v=0

      • 注： 对于映射 L∈L(V,V)L\in \mathcal{L}(V,V)L∈L(V,V) 以及子空间 U⊂VU\sub VU⊂V

        • v∈Uv\in Uv∈U 是 LLL 的特征向量⇔\Leftrightarrow⇔ vvv 是压缩映射 L∣UL|_UL∣U​ 的特征向量

        • v∈Uv\in Uv∈U 是 LLL 的特征向量 ⇔\Leftrightarrow⇔ vvv 是扩展映射 L~\tilde{L}L~ 的特征向量

        • L=L∣U~+L∣U⊥~L = \widetilde{L|_U}+ \widetilde{L|_{U^\perp}}L=L∣U​​+L∣U⊥​​

          • 注中之注：L∣UL|_UL∣U​ 并不是必然成立的, 前提是 满足在 UUU 下 endo

  • 谱定理 Spectral Theorem

    • 内容：有限维内积空间 WWW 中映射 L∈L(V,V)L \in \mathcal{L}(V,V)L∈L(V,V) 是一个 自伴算子，存在正交基是 LLL 的一组特征向量

    • 推论：矩阵 AAA 是自伴算子 ⇒\Rightarrow⇒ AAA 可以对角化

      • 细化结论：由于自伴算子有一组特征向量（实）是正交基，令其组成矩阵 UUU，因此满足 U−1=U∗U^{-1} = U^*U−1=U∗, diag(λ)=U∗AUdiag(\lambda) = U^*AUdiag(λ)=U∗AU
