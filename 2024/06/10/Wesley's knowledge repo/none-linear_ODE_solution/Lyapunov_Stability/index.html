稳定性

研究方程

dydt=g(t,y),y∈Rn\frac{dy}{dt} = g(t,y), y\in \mathbb R^n dtdy​=g(t,y),y∈Rn

稳定和不稳定

我们把邻近的解均趋于之的特解称为稳定，反之称为不稳定

驻定和平衡解

对于等式右边 (不包含 dx/dt) 如果不含 自变量 t, 我们称之为 驻定微分方程，再将 微分项设为 0得到的常数解，称之为 平衡解或者驻定解

研究思路

既然我们要研究特解和标准解的差异，故定义差异函数 x=y−ϕ(t)x = y - \phi(t)x=y−ϕ(t), 其中 ϕ(t)\phi(t)ϕ(t) 表示特解, 稳定函数会有 x→0x\to 0x→0 ; 不稳定函数会有 x→∞x\to\inftyx→∞.
再定义函数 dxdt=f(t,x)\frac{dx}{dt} = f(t,x)dtdx​=f(t,x), 带回原方程得到

f(t,x)=g(t,y)−dϕ(t)dt=g(t,x+ϕ(t))−g(t,ϕ(t))f(t,x) = g(t,y) - \frac{d\phi(t)}{dt} =g(t,x+\phi(t))-g(t,\phi(t)) f(t,x)=g(t,y)−dtdϕ(t)​=g(t,x+ϕ(t))−g(t,ϕ(t))

对于函数 fff, 我们有特解 f(t,0)=0f(t,0) = 0f(t,0)=0, 即稳定性等同于趋于 0 的趋势

定理 1 (线性稳定证明)

对常系数线性微分方程组 x˙=Ax\dot x = Axx˙=Ax 若其特征方程 det⁡(A−λE)=0\det(A - \lambda E) = 0det(A−λE)=0 具有负实部则方程零解渐近稳定; 若具有正实部根，不稳定; 若由零实部根，可能稳定也可能不稳定

定理 2 (非线性算子稳定证明)

对于非线性微分方程组 x˙=Ax+R(x)\dot x = Ax + R(x)x˙=Ax+R(x) 且 R(0)=0,lim⁡∣∣x∣∣→0∣∣R(x)∣∣∣∣x∣∣=0R(0) = 0, \lim_{||x||\to 0} \frac{||R(x)||}{||x||} = 0R(0)=0,lim∣∣x∣∣→0​∣∣x∣∣∣∣R(x)∣∣​=0, 若特征方程没有零实部根，则在方程的零解的稳定性和 x˙=Ax\dot x = Axx˙=Ax 的一致，若有负实部根，稳定; 正实部，不稳定.
