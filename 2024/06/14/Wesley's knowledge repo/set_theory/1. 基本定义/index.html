1. 基本定义

发表于2024-06-14|更新于2024-06-14|math
|阅读量:

等号三律

自反律

对于索伦变量 xxx, 我们有 x=xx = xx=x

对称律

对于变量 x,yx,yx,y, 若我们有 x=yx = yx=y, 则有 y=xy = xy=x

传递律

对于变量 x,y,zx,y,zx,y,z, 如果有 x=y,x=zx = y, x = zx=y,x=z, 则我们有 y=zy = zy=z

同一律

两个集合 xxx 和 yyy 相等的充分必要条件是 他们具有完全相同的元素成员
集合的同一律表示：任何一个集合都有它的 外延 唯一确定

二元关系

这些内容和离散数学的基础有点重合了，详情请参照[文章](…/‘discrete math’/‘逻辑和证明.md’)

集合论的表达式

我们使用五条规则递归的表示集合论

  1. 每一个基本表达式都是一个表达式 比如 x∈yx\in yx∈y
  2. 如果 ϕ\phiϕ 是一个表达式 那么 (¬ϕ)(\neg \phi)(¬ϕ) 是一个表达式
  3. 如果 ϕ1\phi_1ϕ1​, ϕ2\phi_2ϕ2​ 是一个表达式，那么 (ϕ1∧ϕ2),(ϕ1∨ϕ2),(ϕ1→ϕ2),(ϕ1↔ϕ2)(\phi_1\land \phi_2),(\phi_1\lor \phi_2), (\phi_1\to \phi_2),(\phi_1\leftrightarrow \phi_2)(ϕ1​∧ϕ2​),(ϕ1​∨ϕ2​),(ϕ1​→ϕ2​),(ϕ1​↔ϕ2​) 是一个表达式
  4. 如果 ϕ\phiϕ 是一个表达式，那么 ∀xϕ\forall x\phi∀xϕ, ∃yϕ\exists y \phi∃yϕ 都是一个表达式
  5. 任意表达式都只能从上述的四种方式得到

集合等于号的定义

我们使用同一律逻辑定义等于号

∀x∀y((x=y))↔(∀z(z∈x↔z∈y)))\forall x\forall y((x = y))\leftrightarrow (\forall z(z\in x\leftrightarrow z\in y))) ∀x∀y((x=y))↔(∀z(z∈x↔z∈y)))

上述定义公式我们常称之为 公理1，也叫做 外延公理 (extension axiom)

外延 extension 和 内涵 intension

外延：关注对象集合的本身，即对象的实际构成，或者说就是元素，例如 {1,2,3,4}
内涵：关注对象的特性或者定义方式而非具体内容，例如 小于5的自然数集合

约束和自由

我们称被量词修饰的变量是 约束变量，没有被量词修饰的是 自由变量。
