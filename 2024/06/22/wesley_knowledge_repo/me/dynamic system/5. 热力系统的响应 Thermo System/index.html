5. 热力系统的响应 Thermo System

Created2024-06-22|Updated2024-06-22|physics
|Post Views:

参数

  • 温度 T
  • 热量传递速率 q
  • 热阻 R
  • 热容 C

公式

q=1R(T−Ta)q = \frac{1}{R}(T - T_a) q=R1​(T−Ta​)

∑heat in=CsT=qin−qout\sum \text{heat in} = CsT = q_{in} - q_{out} ∑heat in=CsT=qin​−qout​

思路

输入热量 - 输出热量 = 净热量

qin−qout=qnet=Cs(T−0)=CsTq_{in} - q_{out} = q_{net} = Cs(T - 0) = CsT qin​−qout​=qnet​=Cs(T−0)=CsT

输出热量 = 温差 / 热阻

qout=1R(T−Tenviron)q_{out} = \frac{1}{R}(T - T_{\text{environ}}) qout​=R1​(T−Tenviron​)

结合得到动态方程

CT˙+1RT=qin+1RTenvironC\dot T + \frac{1}{R}T = q_{in} + \frac{1}{R}T_{\text{environ}} CT˙+R1​T=qin​+R1​Tenviron​
