4. 循环群 Cyclic Groups

Created2024-07-20|Updated2024-07-20|math
|Post Views:

定义

一个群是循环的如果可以由单个元素生成得到

例子

对于一个运算符是 ×\times× 的群，我们有 x∈Gx\in Gx∈G 的子群为 H:={⋯ ,x−3,x−2,x−1,1=x0,x,x2,x3}H:= \{\cdots,x^{-3},x^{-2},x^{-1},1=x^0,x,x^2,x^3\}H:={⋯,x−3,x−2,x−1,1=x0,x,x2,x3}
这个群就是包含 xxx 的最小子群，写作 ⟨x⟩\langle x\rangle⟨x⟩
如果存在 m∈N−{0}m\in \mathbb N - \{0\}m∈N−{0} 使得 xm=1x^m = 1xm=1 我们称 mmm 是 order of x, 写作 m=∣x∣m = |x|m=∣x∣
这个 m 将一个发散的群结构变成了循环的群结构，并且说明了群的尺寸
对于群 GGG 的 order，我们有 ∣G∣|G|∣G∣ 为其中元素的数量

注

这里的 xnx^nxn 可能表示不同的元素，或者相同，如 (−1)m(-1)^m(−1)m

定理

令 ⟨x⟩\langle x\rangle⟨x⟩ 是一个循环子群，元素由 xxx 生成，令 S:={k∈Z∣xk=1}S:=\{k\in \mathbb Z | x^k = 1\}S:={k∈Z∣xk=1} 那么

  1. 集合 SSS 是加法群 (Z,+)(\mathbb Z, +)(Z,+) 的子群
    1. 令 k,l∈Sk,l\in Sk,l∈S 那么 xk=xl=1x^k = x^l = 1xk=xl=1 因此我们有 xk+l=xkxl=1x^{k+l} = x^k x^l = 1xk+l=xkxl=1 因此 k+l∈Sk+l\in Sk+l∈S
    2. x0=1x ^0 = 1x0=1 所以 0∈S0\in S0∈S
    3. 如果 k∈Sk\in Sk∈S 那么 x−k=(xk)−1=1x^{-k} = (x^k)^{-1} = 1x−k=(xk)−1=1 因此 k∈Sk\in Sk∈S
  2. 对于 r,s∈Zr,s\in \mathbb Zr,s∈Z, xr=xsx^r = x^sxr=xs 当且仅当 xr−s=1x^{r-s} = 1xr−s=1 即 r−s∈Sr-s\in Sr−s∈S
  3. 假设 S≠{0}S\not =\{0\}S​={0} 那么 S=nZS = n\mathbb ZS=nZ 对某些 n∈N\{0}n\in\mathbb N\backslash \{0\}n∈N\{0} , 且 1,x,⋯ ,xn1,x,\cdots,x^n1,x,⋯,xn 是不同元素，且 ∣⟨x⟩∣=n|\langle x\rangle| = n∣⟨x⟩∣=n
    1. 由于 SSS 是 (Z,+)(\mathbb Z, +)(Z,+) 的一个子群，那么 S=nZS = n\mathbb ZS=nZ 对某些最小的正整数 n∈Sn\in Sn∈S 存在对于任意 k∈Zk\in \mathbb Zk∈Z, k=qn+rk=qn + rk=qn+r 对某些 q 与 r 成立，因此 xk=xqn+r=xnqxr=xrx^k = x^{qn+r} = x^{nq}x^r = x^rxk=xqn+r=xnqxr=xr

注

  • 如果 ∣x∣=∞|x| = \infty∣x∣=∞ 那么 xr=xsx^r = x^sxr=xs 当且仅当 r=sr = sr=s , 因为这个群没有循环起来
  • 如果 ∣x∣<∞|x|< \infty∣x∣<∞ 或者说 ∣x∣=n∈N|x| = n\in \mathbb N∣x∣=n∈N 那么 xr=xsx^r = x^sxr=xs 当且仅当 n∣r−sn | r - sn∣r−s
  • ∣x∣|x|∣x∣ = ∣⟨x⟩∣|\langle x\rangle|∣⟨x⟩∣
  • 如果 ∣x∣=n|x| = n∣x∣=n 且 xk=1x^k = 1xk=1 那么 n ∣ kn\ |\ kn ∣ k

最大公约数定理

令 n,k∈N\{0}n,k\in\mathbb N\backslash\{0\}n,k∈N\{0} 对于群 GGG 和 x∈Gx\in Gx∈G 有 ∣x∣=n\{0}|x| = n\backslash\{0\}∣x∣=n\{0}
那么 ⟨xk⟩=⟨xgcd(n,k)⟩\langle x^k\rangle = \langle x^{\text{gcd(n,k)}}\rangle⟨xk⟩=⟨xgcd(n,k)⟩ 且有 ∣xk∣=ngcd(n,k)|x^k| = \frac{n}{\text{gcd(n,k)}}∣xk∣=gcd(n,k)n​

证明

因为有 ∣x∣=n|x| = n∣x∣=n 我们有

⟨xk⟩={(xk)t∣t∈Z}={xkt+ns∣t,s∈Z}={xd∣d∈gcd(n,k)Z}=⟨xgcd(n,k)⟩\langle x^k \rangle= \{(x^k)^t|t\in \mathbb Z\} = \{x^{kt + ns}| t,s\in \mathbb Z\} = \{x^d | d\in gcd(n,k)\mathbb Z\} = \langle x^{\text{gcd(n,k)}}\rangle ⟨xk⟩={(xk)t∣t∈Z}={xkt+ns∣t,s∈Z}={xd∣d∈gcd(n,k)Z}=⟨xgcd(n,k)⟩

对于 ∣xk∣|x^k|∣xk∣ 我们其实就是找 (xk)r=1(x^k)^r = 1(xk)r=1 的最小 r 值
那么对于 t=∣xk∣t = |x^k|t=∣xk∣, 我们有 t=∣xk∣=∣⟨xk⟩∣=∣⟨xgcd(n,k)⟩∣=∣xgcd(n,k)∣t = |x^k| = |\langle x^k\rangle| = | \langle x^{\text{gcd(n,k)}}\rangle| = |x^\text{gcd(n,k)}|t=∣xk∣=∣⟨xk⟩∣=∣⟨xgcd(n,k)⟩∣=∣xgcd(n,k)∣ 因此

  • (xk)t=1⇔xkt=1⇒n∣kt⇔ngcd(n,k) ∣ t(x^k)^t = 1 \Leftrightarrow x^{kt} = 1 \Rightarrow n | kt \Leftrightarrow \frac{n}{\text{gcd(n,k)}}\ |\ t(xk)t=1⇔xkt=1⇒n∣kt⇔gcd(n,k)n​ ∣ t
  • (xgcd(n,k))n/gcd(n,k)=xn=1⇒t ∣ ngcd(n,k)(x^{\text{gcd(n,k)}})^{\text{n/gcd(n,k)}} = x^n = 1 \Rightarrow t\ |\ \frac{n}{\text{gcd(n,k)}}(xgcd(n,k))n/gcd(n,k)=xn=1⇒t ∣ gcd(n,k)n​

注

  1. 令 ∣⟨x⟩∣<∞|\langle x\rangle | < \infty∣⟨x⟩∣<∞ 那么 y∈⟨x⟩⇒∣y∣ divides ∣⟨x⟩∣y\in \langle x\rangle \Rightarrow|y|\ divides\ |\langle x\rangle|y∈⟨x⟩⇒∣y∣ divides ∣⟨x⟩∣
    1. 这其实就是把 y=xky = x^ky=xk 带入等式
  2. 令 ∣x∣=n∈N\{0}|x| = n \in \mathbb N \backslash \{0\}∣x∣=n∈N\{0} 那么我们有 ⟨xi⟩=⟨xj⟩⇔∣xi∣=∣xj∣⇔gcd(n,i)=gcd(n,j)\langle x^i\rangle = \langle x^j \rangle \Leftrightarrow |x^i| = |x^j| \Leftrightarrow \text{gcd(n,i)} = \text{gcd(n,j)}⟨xi⟩=⟨xj⟩⇔∣xi∣=∣xj∣⇔gcd(n,i)=gcd(n,j)
