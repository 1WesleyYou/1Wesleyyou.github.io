5. 同态 Homomorphism

Created2024-07-24|Updated2024-07-30|math
|Post Views:

为什么我们要在学习群的时候突然探讨homomorphism？同态提供了两个群空间下相似的映射特性，那么如果我知道一个群的性质（循环？对称？）那么另一个就会具有相似的形式.
同样的，如果我们想知道子群和总群的关系，尤其是有限群的性质，我们可以联想到线性空间的研究方法: 利用线性无关的基将线性空间表述为集合基的线性组合。类似的，对于群的运算符，我们联想到使用循环群的思路，我们可以将一个总循环群变成几个循环群的并集(或者说 span?) 如果能做到这样，那么我们的空间就更容易理解。
当然，线性空间的问题是，其具有无限性，但是对于集合而言，我们更加应该看中其有限和无限的特殊性质，对于有限群，那么如果能分割称几个子群，那么其 order 的关系是什么？

定义

对于群 G,G′G,G'G,G′ 我们有同态f使得

f(xy)=f(x)f(y)f(xy) = f(x)f(y) f(xy)=f(x)f(y)

注

  1. 简单同态 f:x→1G′f: x\to 1_{G'}f:x→1G′​
  2. 包含映射 inclusion map: H→GH\to GH→G 且 x→xx\to xx→x 当 HHH 是 GGG 的一个子群
  3. 行列式函数 det⁡:GLn(R)→Rx\det: GL_n(\mathbb R)\to \mathbb R^xdet:GLn​(R)→Rx
  4. 正负映射 sgn: Sn→{±1}S_n\to \{\pm 1\}Sn​→{±1}
  5. 指数映射 exp: (R,+)→Rx,x→ex(\mathbb R,+)\to \mathbb R^x,x\to e^x(R,+)→Rx,x→ex
  6. 绝对值映射 ∣⋅∣|\cdot|∣⋅∣: Cx→Rx\mathbb C^x \to \mathbb R^xCx→Rx
  7. 对于函数 Z→S2\mathbb Z\to S_2Z→S2​ 偶数 →e\to e→e 奇数 →τ\to \tau→τ

群同态定理

对于函数 f:G→G′f:G\to G'f:G→G′ 是一个群同态，那么

  1. 如果 a1,⋯an∈Ga_1,\cdots a_n\in Ga1​,⋯an​∈G 那么 f(a1,⋯ ,an)=f(a1)f(a2)⋯f(an)f(a_1,\cdots,a_n) = f(a_1)f(a_2)\cdots f(a_n)f(a1​,⋯,an​)=f(a1​)f(a2​)⋯f(an​)
  2. f(1G)=1G′f(1_G) = 1_{G'}f(1G​)=1G′​
  3. f(a−1)=f(a)−1f(a^{-1}) = f(a)^{-1}f(a−1)=f(a)−1
    从这些定理可以看出，如果 HHH 是 GGG 的一个子群，那么 f(H)f(H)f(H) 是 G′G'G′ 的一个子群

群同态的像与核

imf:={x∈G′ ∣ x=f(a),a∈G}\text{im} f:=\{x\in G'\ |\ x = f(a), a\in G\}imf:={x∈G′ ∣ x=f(a),a∈G}
kernel f:={a∈G ∣ f(a)=1G′}\text{kernel}\ f:=\{a\in G\ |\ f(a) = 1_{G'}\}kernel f:={a∈G ∣ f(a)=1G′​}

例子

  1. 行列式的核是 SLn(R)SL_n(\mathbb R)SLn​(R) 即行列式为1的nn矩阵
  2. 符号同态 sgn: Sn→{±1}S_n\to \{\pm 1\}Sn​→{±1} 其核为 AnA_nAn​ 即交替群，或者说是偶置换的数量

陪集 Coset

对于一个群 G, 如果有 H≤GH\le GH≤G 且 a∈Ga\in Ga∈G 那么符号 aHaHaH 就是所有ah的积

aH={g∈G ∣ g=ah,h∈H}aH = \{g\in G\ |\ g = ah,h\in H\} aH={g∈G ∣ g=ah,h∈H}

那么这个集合就称为 HHH 的左陪集

核空间陪集的等价性

对于函数 f:G→G′f: G\to G'f:G→G′ 是 homomorphism 并且令 a,b∈Ga,b\in Ga,b∈G 且令 K=ker⁡fK = \ker fK=kerf 那么以下四个性质等价:

  1. f(a)=f(b)f(a) = f(b)f(a)=f(b)
  2. a−1b∈Ka^{-1}b\in Ka−1b∈K
  3. b∈aKb\in aKb∈aK
  4. aK=bKaK = bKaK=bK

推论

群同态 f:G→G′f:G\to G'f:G→G′ 单射当且仅当 ker⁡f={1G}\ker f = \{1_G\}kerf={1G​}

注

对于 homomorphism 的核空间的左陪集可以将其看作是等价类，每一个等价类的 f 映射结果相同 即 每个核子空间等价类和 Im fIm\ fIm f 一一对应

对于一个子群而言，我们定义以其陪集划分的数量， 写作 [G:H][G:H][G:H]
这个表示总集 GGG 的 order 是 HHH 的 order 的倍数

计数公式 Counting Formula

由于所有左陪集的尺寸相同且其划分了总群 GGG, 因此有等式

∣G∣=∣H∣⋅[G:H]|G| = |H|\cdot [G:H] ∣G∣=∣H∣⋅[G:H]

拉格朗日定理 Lagrange’s Theorem

对于总群 GGG 的子群 HHH 我们都有 HHH 的 order 整除 GGG 的 order

  • 注意这里的逻辑方向，是说如果有子群，那么order整除; 但是没有说明一定有因数order的子群存在性。但是机智的数学家证明了对于p-有限群（order为 pk∈Pp^k\in \mathbb Ppk∈P) 那么一定有非平凡子群(trivial, 即只有单位元的群)

推论

有限群的元素的 order 整除整个群的 order

注

这里的 order 指的是以该元素为生成元的循环子群
而且由于 HHH 是子群所以用 counting formula 就可以说明整除性

推论 3

对于有限群 G,G′G,G'G,G′, 我们有 homomorphism f:G→G′f:G\to G'f:G→G′ 那么

  1. ∣G∣=∣ker⁡f∣⋅∣imf∣|G| = |\ker f| \cdot |\text{im} f|∣G∣=∣kerf∣⋅∣imf∣
  2. ∣ker⁡f∣|\ker f|∣kerf∣ 整除 ∣G∣|G|∣G∣
  3. ∣imf∣|\text{im} f|∣imf∣ 整除 ∣G′∣|G'|∣G′∣ 和 ∣G∣|G|∣G∣

证明

对于群 GGG 我们有 子群 ker⁡f\ker fkerf ，对这个子群求左陪集得到的等价类和 Im fIm\ fIm f 一一对应，所以有 ∣G∣=∣ker⁡f∣⋅∣imf∣|G| = |\ker f| \cdot |\text{im} f|∣G∣=∣kerf∣⋅∣imf∣
