定义

A(x)=∑n≥0anxnA(x) = \sum_{n \ge 0} a_n x^n A(x)=n≥0∑​an​xn

  • 这是一个 generating function 生成函数
  • 我们称 x 是变量 variable 或者 indeterminate
  • 如果要获得 x 的值，我们需要用序列 an=(0,1,0,⋯ )a_n = ( 0, 1, 0,\cdots )an​=(0,1,0,⋯) 来进行获得
  • 我们也可以写作 [xn]A(x)=an[x^n]A(x) = a_n[xn]A(x)=an​

性质

对于形式幂级数 A(x)=∑n≥0anxnA(x) = \sum_{n\ge 0}a_n x^nA(x)=∑n≥0​an​xn, B(x)=∑n≥0bnxnB(x) = \sum_{n\ge 0}b_n x^nB(x)=∑n≥0​bn​xnm C(x)=∑n≥0cnxnC(x) = \sum_{n\ge 0}c_n x^nC(x)=∑n≥0​cn​xn

  1. 等价 equality: A(x)=B(x)⇔an=bnA(x) = B(x) \Leftrightarrow a_n = b_nA(x)=B(x)⇔an​=bn​
  2. 加法 addition: A(x)+B(x)=∑n≥0(an+bn)xnA(x) + B(x) = \sum_{n\ge 0}(a_n + b_n)x^nA(x)+B(x)=∑n≥0​(an​+bn​)xn
    1. 交换律 A(x)+B(x)=B(x)+A(x)A(x) + B(x) = B(x) + A(x)A(x)+B(x)=B(x)+A(x)
    2. 结合律 (A(x)+B(x))+C(x)=A(x)+(B(x)+C(x))(A(x) + B(x)) + C(x) = A(x) + (B(x) + C(x))(A(x)+B(x))+C(x)=A(x)+(B(x)+C(x))
    3. 零元 A(x)+0=A(x)A(x) + 0 = A(x)A(x)+0=A(x)
    4. 相反元 A(x)+(−A(x))=0A(x) + (-A(x)) = 0A(x)+(−A(x))=0
  3. 乘法 multiplication: A(x)B(x)=∑n≥0(∑k=0nakbn−k)xnA(x)B(x) = \sum_{n\ge 0}(\sum_{k=0}^n a_k b_{n-k})x^nA(x)B(x)=∑n≥0​(∑k=0n​ak​bn−k​)xn
  4. 交换律 A(x)B(x)=B(x)A(x)A(x)B(x) = B(x)A(x)A(x)B(x)=B(x)A(x)
  5. 结合律 (A(x)B(x))C(x)=A(x)(B(x)C(x))(A(x)B(x))C(x) = A(x)(B(x)C(x))(A(x)B(x))C(x)=A(x)(B(x)C(x))
  6. 单位元 A(x)1=A(x)A(x)1 = A(x)A(x)1=A(x), 1=a+0x+0x2⋯1 = a + 0x + 0x^2 \cdots1=a+0x+0x2⋯
  7. 分配律 A(x)(B(x)+C(x))=A(x)B(x)+A(x)C(x)A(x)(B(x) + C(x)) = A(x)B(x) + A(x)C(x)A(x)(B(x)+C(x))=A(x)B(x)+A(x)C(x)
    整体来说，形式幂级数是一个交换环
  8. 可逆性 invertibility: 如果 A(x)B(x)=1A(x)B(x) = 1A(x)B(x)=1, 那么 我们说 A 是 invertible, 其中B(x)B(x)B(x) 并不需要是形式幂级数，只是一个函数
    写法: B(x)=A(x)−1=1A(x)B(x) = A(x)^{-1} = \frac{1}{A(x)}B(x)=A(x)−1=A(x)1​
