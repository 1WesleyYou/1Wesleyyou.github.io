2. еҠ жі•еҷЁпјҢеҮҸжі•еҷЁе’Ңд№ҳжі•еҷЁ

Created2024-07-05|Updated2026-03-26|cs_basic
|Post Views:

е…ЁеҠ еҷЁ

иҖғиҷ‘иҝӣдҪҚзҡ„еҠ жі•з§°дёәе…ЁеҠ пјҢиғҪеӨҹе®ҢжҲҗе…ЁеҠ иҝҗз®—зҡ„з”өи·Ҝз§°дёәе…ЁеҠ еҷЁпјҢдёҖдёӘеҹәжң¬е…ЁеҠ еҷЁиғҪеӨҹе®ҢжҲҗдёӨдёӘдёҖдҪҚдәҢиҝӣеҲ¶ж•°зҡ„е…ЁеҠ иҝҗз®—пјҢеӣ жӯӨе®ғе…·жңүдёүдёӘиҫ“е…Ҙз«Ҝе’ҢдёӨдёӘиҫ“еҮәз«ҜпјҢе…¶дёӯ Xi,YiX_i, Y_iXiвҖӢ,YiвҖӢ дёәеҠ ж•°пјҢCiвҲ’1C_{i - 1}CiвҲ’1вҖӢ зӣёйӮ»дҪҺдҪҚиҝӣжқҘзҡ„иҝӣдҪҚж•°пјҢ SiS_iSiвҖӢ дёәиҫ“еҮәе’ҢпјҢCiC_iCiвҖӢ дёәеҗ‘й«ҳдҪҚйӮ»дҪҚиҝӣдҪҚж•°

е’ҢдҪҚ

жҲ‘д»¬з§° s жҳҜе’ҢдҪҚиҫ“еҮәпјҢи®Ўз®—ж–№жі•жҳҜеҜ№дәҺ x_i, y_i c_i-1 зҡ„ејӮжҲ–з»“жһң

s=xвҠ•yвҠ•cs = x\oplus y\oplus c s=xвҠ•yвҠ•c

йӮЈд№ҲеңЁе®һйҷ…з”өи·ҜдёӯжҲ‘д»¬еҸӘжңүдәҢе…ғиҫ“е…Ҙзҡ„ејӮжҲ–й—ЁпјҢйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»Ҙз”ЁдёӨдёӘејӮжҲ–й—ЁиҝӣиЎҢз»„еҗҲе®ҢжҲҗ

иҝӣдҪҚдҪҚ

еңЁдёүдёӘеҸҳйҮҸдёӯжңүдёӨдёӘеҸҳйҮҸжҳҜ1зҡ„ж—¶еҖҷеҸ‘з”ҹиҝӣдҪҚ

c=(xвӢ…y)+(xвӢ…c)+(yвӢ…c)c = (x\cdot y) + (x\cdot c) + (y\cdot c) c=(xвӢ…y)+(xвӢ…c)+(yвӢ…c)

иҖғиҷ‘еҲ°жҲ‘д»¬йңҖиҰҒдҪҝз”Ёе°ҪйҮҸе°‘зҡ„йҖ»иҫ‘й—ЁпјҢйӮЈд№ҲжҲ‘д»¬йңҖиҰҒеҜ№иҝҷдёӘйҖ»иҫ‘иЎЁиҫҫејҸиҝӣиЎҢеҢ–з®Җ

c=xвӢ…y+xвӢ…c+yвӢ…c=xвӢ…y+cвӢ…(x+y)c = x\cdot y + x\cdot c + y\cdot c = x\cdot y + c\cdot(x + y) c=xвӢ…y+xвӢ…c+yвӢ…c=xвӢ…y+cвӢ…(x+y)

йӮЈд№ҲжҲ‘д»¬еңЁverilogдёӯеҸҜд»Ҙе®һзҺ°иҝҷдёӘд»Јз Ғ, жҲ‘д»¬йңҖиҰҒз”ЁеҲ°дёҺй—Ёе’ҢжҲ–й—ЁеӨ„зҗҶ

еҚҠеҠ еҷЁ half adder

з”ЁдәҺе°ҶдёӨдёӘеҚ•дёӘдҪҚдәҢиҝӣеҲ¶ж•°зӣёеҠ пјҢе№¶з”ҹжҲҗдёҖдёӘе’ҢдҪҚпјҲsumпјүе’ҢдёҖдёӘиҝӣдҪҚдҪҚпјҲcarryпјүгҖӮдёҺе…ЁеҠ еҷЁдёҚеҗҢзҡ„жҳҜпјҢеҚҠеҠ еҷЁдёҚеӨ„зҗҶиҫ“е…Ҙзҡ„иҝӣдҪҚдҪҚпјҢеҸӘеӨ„зҗҶдёӨдёӘиҫ“е…ҘдҪҚгҖӮ

  • еҚҠеҠ еҷЁ з”ЁдәҺз®ҖеҚ•зҡ„еҚ•дёӘдҪҚеҠ жі•пјҢжІЎжңүиҝӣдҪҚиҫ“е…ҘпјҢйҖӮз”ЁдәҺжңҖдҪҺдҪҚеҠ жі•е’ҢеҹәзЎҖжһ„е»әжЁЎеқ—гҖӮ
  • е…ЁеҠ еҷЁ з”ЁдәҺеӨ„зҗҶеӨҡдҪҚдәҢиҝӣеҲ¶еҠ жі•зҡ„жүҖжңүдҪҚиҝҗз®—пјҢиғҪеӨҹеӨ„зҗҶжқҘиҮӘдҪҺдҪҚзҡ„иҝӣдҪҚиҫ“е…ҘпјҢйҖӮз”ЁдәҺеӨҚжқӮеҠ жі•з”өи·Ҝе’ҢеёҰиҝӣдҪҚиҫ“е…Ҙзҡ„еңәжҷҜгҖӮ

еҮҸжі•еҷЁ

еҚҠеҮҸеҷЁпјҲHalf Subtractorпјү

еҚҠеҮҸеҷЁ з”ЁдәҺеҮҸеҺ»дёӨдёӘеҚ•дёӘдҪҚе№¶з”ҹжҲҗе·®еҖјпјҲdifferenceпјүе’ҢеҖҹдҪҚпјҲborrowпјүгҖӮе®ғжңүдёӨдёӘиҫ“е…Ҙе’ҢдёӨдёӘиҫ“еҮәгҖӮ

  • иҫ“е…Ҙпјҡ дёӨдёӘдәҢиҝӣеҲ¶дҪҚ x е’Ң yгҖӮ
  • иҫ“еҮәпјҡ дёҖдёӘе·®еҖјдҪҚ d е’ҢдёҖдёӘеҖҹдҪҚдҪҚ b

е·®еҖјдҪҚ

дёӨдёӘиҫ“е…ҘдҪҚзҡ„е·®еҖјпјҢз”ЁдәҺиЎЁзӨә x - yгҖӮ

d=xвҠ•yd = x \oplus y d=xвҠ•y

еҖҹдҪҚдҪҚ

з”ЁдәҺиЎЁзӨә x - y зҡ„еҖҹдҪҚпјҢеҪ“ x < y ж—¶пјҢйңҖиҰҒеҖҹдҪҚгҖӮ

b=xвҖҫвӢ…yb = \overline{x} \cdot y b=xвӢ…y

е…¶дёӯ xЛү\bar xxЛү иЎЁзӨәеҜ№xйҖ»иҫ‘еҸ–еҸҚ

е…ЁеҮҸеҷЁпјҲFull Subtractorпјү

е·®еҖјдҪҚ

d=xвҠ•yвҠ•bd = x \oplus y \oplus b d=xвҠ•yвҠ•b

еҖҹдҪҚдҪҚ

b=(xвҖҫвӢ…y)+((xвҠ•yвҖҫ)вӢ…b)b = (\overline{x} \cdot y) + ((\overline{x\oplus y}) \cdot b) b=(xвӢ…y)+((xвҠ•yвҖӢ)вӢ…b)

д№ҳжі•еҷЁ multiplier

йҖҡиҝҮйҖҗдҪҚд№ҳжі•е’Ңз§»дҪҚж“ҚдҪңжқҘе®һзҺ°дәҢиҝӣеҲ¶ж•°зҡ„д№ҳз§Ҝ

  • йҖҗдҪҚд№ҳжі•пјҡе°Ҷиў«д№ҳж•°дёҺд№ҳж•°зҡ„жҜҸдёҖдҪҚзӣёд№ҳгҖӮ
  • з§»дҪҚе’ҢзҙҜеҠ пјҡе°ҶйҖҗдҪҚд№ҳжі•зҡ„з»“жһңиҝӣиЎҢз§»дҪҚпјҢе№¶е°ҶжүҖжңүйғЁеҲҶз§ҜзҙҜеҠ иө·жқҘпјҢеҫ—еҲ°жңҖз»Ҳзҡ„д№ҳз§ҜгҖӮ

еҹәжң¬дәҢиҝӣеҲ¶д№ҳжі•еҷЁ

еҹәжң¬дәҢиҝӣеҲ¶д№ҳжі•еҷЁжҳҜжңҖз®ҖеҚ•зҡ„дёҖз§Қд№ҳжі•еҷЁпјҢе®ғеҹәдәҺйҖҗдҪҚд№ҳжі•е’Ңз§»дҪҚж“ҚдҪңгҖӮд»ҘдёӢжҳҜе…¶еҹәжң¬е®һзҺ°ж–№жі•пјҡ
еҒҮи®ҫжҲ‘д»¬жңүдёӨдёӘ4дҪҚдәҢиҝӣеҲ¶ж•° A е’Ң Bпјҡ A=A3A2A1A0A=A_3A_2A_1A_0A=A3вҖӢA2вҖӢA1вҖӢA0вҖӢвҖӢ, B=B3B2B1B0B=B_3B_2B_1B_0B=B3вҖӢB2вҖӢB1вҖӢB0вҖӢвҖӢ

еҹәжң¬дәҢиҝӣеҲ¶д№ҳжі•еҷЁзҡ„жӯҘйӘӨ

  1. йҖҗдҪҚд№ҳжі•пјҡ
    • и®Ўз®— A дёҺ B жҜҸдёҖдҪҚзҡ„д№ҳз§ҜпјҢеҫ—еҲ°йғЁеҲҶз§ҜпјҲPartial ProductsпјүгҖӮ
  2. з§»дҪҚе’ҢзҙҜеҠ пјҡ
    • еҜ№жҜҸдёҖдёӘйғЁеҲҶз§ҜиҝӣиЎҢз§»дҪҚж“ҚдҪңпјҢдҪҝе…¶еҜ№йҪҗгҖӮ
    • е°ҶжүҖжңүйғЁеҲҶз§ҜзҙҜеҠ иө·жқҘпјҢеҫ—еҲ°жңҖз»Ҳд№ҳз§ҜгҖӮ

P=AГ—B=вҲ‘i=0nвҲ’1(BiГ—A<<i)P = A\times B = \sum_{i = 0}^{n-1}(B_i \times A << i) P=AГ—B=i=0вҲ‘nвҲ’1вҖӢ(BiвҖӢГ—A<<i)

зұ»дјјдәҺеҲ—з«–ејҸзҡ„д№ҳжі•з®—жі•

з§»дҪҚж“ҚдҪң Shifter

{} жҳҜиҝһжҺҘж“ҚдҪңз¬Ұ
иҝҷйҮҢзҡ„ 1'b0 иЎЁзӨә 1 дҪҚзҡ„дәҢиҝӣеҲ¶ж•° 0
A[2:0] иЎЁзӨә 3 дҪҚж•°жҚ®дҝЎеҸ· A
assign иЎЁзӨәиҝһз»ӯиөӢеҖј

е®һзҺ°д№ҳжі•

иҙҹж•°дёҺиЎҘз Ғ

еҜ№дәҺ1еӯ—иҠӮж•°жҚ®пјҢеҰӮжһңжҲ‘д»¬иҰҒиЎЁзӨәиҙҹж•°пјҢйӮЈд№ҲжҲ‘д»¬е°ұиҰҒе°ҶдёҖдҪҚж•°жҚ®иҪ¬еҸҳжҲҗз¬ҰеҸ·дҪҚпјҢеңЁе·ҘзЁӢдёҠжҲ‘д»¬дјҡе°ҶжңҖй«ҳдҪҚ1и®ӨдёәжҳҜ иҙҹж•°
йӮЈд№ҲпјҢд»Һ еҚ•зәҜдәҢиҝӣеҲ¶ж•°иҪ¬жҚўдёә 10 иҝӣеҲ¶еәҰж•°зҡ„ж—¶еҖҷпјҢжҲ‘д»¬еҸҜд»ҘзҗҶи§Јдёә 128 -> -128

иЎҘз Ғ

д»ҺдёҠйқўзҡ„е®ҡд№үжҲ‘д»¬еҸҜд»Ҙж„ҹеҸ—еҲ°пјҢеҰӮжһңиҰҒиЎЁзӨә -1, жҲ‘д»¬иҰҒз”ЁзӯүејҸ вҲ’1=вҲ’128+64+32+16+8+4+2+1-1 = -128 + 64 + 32 + 16 + 8 + 4 + 2 + 1вҲ’1=вҲ’128+64+32+16+8+4+2+1 жқҘиЎЁзӨә, дәҢиҝӣеҲ¶иЎЁзӨәдёә 111111111111111111111111 иҖҢе…¶зӣёеҸҚж•° 1 зҡ„дәҢиҝӣеҲ¶иЎЁзӨәдёә 000000010000000100000001
йӮЈд№ҲпјҢжҖҺд№Ҳз”ЁжӯЈж•°жқҘиҪ¬жҚўдёә иҙҹж•°пјҲзӣёеҸҚж•°пјүе‘ў
жҲ‘д»¬е…Ҳе°қиҜ•жүҫжүҫи§„еҫӢзңӢзңӢ
